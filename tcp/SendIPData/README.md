# 概要
TCP synパケットを送付するサンプルです。


# ポイント
IP_HDRINCLオプションを指定することによってIPヘッダもプログラム側で定義して含めることが可能となる。
受信時には、 IP ヘッダーは常にパケットに含まれている。

# 詳細
192.168.1.2から192.168.0.1:80へ送信しています。
```
$ gcc sendsyn.c
$ ./a.out
```

tcpdumpで検知すると次のような出力になります。
```
$ sudo tcpdump port 80
... 
00:36:26.366000 IP 192.168.1.2.search-agent > 192.168.0.1.http: Flags [S], seq 0:26, win 5840, length 26: HTTP
```
